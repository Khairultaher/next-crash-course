### Next.js Project initiation
```
npx create-next-app next-crash-course --use-npm
cd next-crash-course
```
### Run the development server:
```bash
npm run dev
```
### Build for production AND generate static website into "out" folder:
```bash
npm run build
```
### Run in production:
```bash
npm start
```
### API Routes:
```
GET /api/articles
GET /api/articles/:id
```
